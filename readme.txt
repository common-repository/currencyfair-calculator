== Installation ==

To install the plugin follow the following steps:

1. Upload the entire `currencyfair-calculator` folder to the `/wp-content/plugins/` directory or install it via Wordpress Plugins.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Click on the `Settings` link from the 'Plugins' screen or the `CurrencyFair Calculator` admin menu button to configure the plugin. You might need to contact us for this step to obtain a personal 'Secret Key'.

All widgets are in Widgets in your WordPress admin panel.
